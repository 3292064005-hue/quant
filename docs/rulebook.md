# 交易规则说明

## 已落地规则

### 1. 最小申报单位
- 主板：100 股
- 科创板：200 股
- 卖出清仓允许不足一手残股一次性卖出

### 2. 常规涨跌幅推断
- 主板：10%
- ST：5%
- 创业板 / 科创板：20%

### 3. 基础交易可用性
- 停牌不可交易
- 涨停禁止买入（可配置关闭）
- 跌停禁止卖出（可配置关闭）
- ST 买入限制（可配置关闭）

### 4. 组合级风控
- 最大单笔金额
- 单票最大目标仓位
- 顺序现金预留
- 卖单释放现金支持后续买单
- 可卖数量限制
- kill switch
- 黑名单

## 当前边界

### 1. 无涨跌幅限制日
系统优先使用数据源显式给出的涨跌停价 / 状态。
若数据源无法提供，则只做常规比例推断，不宣称已覆盖上市首日等全部例外情形。

### 2. 调仓价格
当前仅支持 `close` 调仓模式。
若配置其他模式，系统会在启动组装阶段直接拒绝启动。

### 3. Slippage
执行层已支持按 `slippage_bps` 做买卖方向差异化滑点：
- 买单：`close * (1 + slippage)`
- 卖单：`close * (1 - slippage)`
