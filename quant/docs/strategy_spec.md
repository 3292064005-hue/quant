# 策略说明

## 默认策略

当前默认策略为 `TopNMomentumStrategy`：

- `lookback`：动量观察窗口
- `top_n`：等权持有前 N 名
- `holding_days`：满足最小历史要求后的调仓周期

## 调仓逻辑

- 策略本身只负责输出目标仓位
- `BacktestEngine` 负责决定某个可调仓日是否真正触发调仓
- 非调仓日不会因为“目标为空”而自动清仓

## 输入约束

策略输入的 `history_by_symbol` 已经由引擎按当前交易日过滤为历史有效证券池。

这意味着策略层默认不再承担以下职责：
- 剔除未来上市证券
- 剔除已退市证券
- 剔除当日无 bar 的证券

## 当前限制

- 当前仅内置一个动量策略
- 暂未引入行业中性、基准跟踪、因子暴露控制等高级组合层逻辑
